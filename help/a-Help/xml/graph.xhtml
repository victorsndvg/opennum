Graph visualization

An element struct can create a graph visualization in the Graphic scene. To do so, the element must have

  • the attribute data to know which file describes the graph,
  • the attribute plot="2d_graph" to indicate that the know which graphic type will be plotted.

The attribute data

An element struct can refer to graph file with the attribute data="prefix:value", where prefix can be

  • file and then value must be a file containing a graph that comply one of the valid formats; value must follow the rules explained in the section File dependencies; 
    <struct name="2D graph" data="file:example.gr2" plot="2d_graph">
</leaf>
  • menu and then value must be a menu path that points to a element leaf with attributes type="file". value must follow the rules explained in the section Menu dependencies. 
    <struct name="2D graph" data="menu:/Files/graph" plot="2d_graph">
</leaf>
