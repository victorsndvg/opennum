Embedded help

On this page:

  • External url
  • Local url
  • HTML data
  • HTML data with links in default browser
  • Tooltip

The application Intermediate - embedded help shows how tocreate embedded help buttons in any graphical widget and how to create help messages printed in labels. It also explains the use of helpdata, helpconfig and tooltip attributes.

This attributes, helpdata, helpconfig and tooltip, can be used either on struct or leaf tags. The value of helpconfig attribute has three components, and are assigned as follows: helpconfig="modifier,width,height", being pixels the units of height and width.

The XML file of this example can be found at <installdir>/apps/k-Intermediate - embedded help/a-New/local.mnu.xml, where <installdir> is the installation folder.


Figure 1: Help button.

External url

With the url modifier assigned to the attribute helpconfig, OpenNum open a HTML window that shows an internet URL. In this case the value of the attribute helpdata must be a valid URL with the HTTP protocol. 

<struct name="External url" helpconfig="url,800,640" helpdata="http://www.usc.es">
    ...
</struct>

Local url

With the urlinst modifier assigned to the attribute helpconfig, the HTML window instances a local URL. In this case the value of the attribute helpdata must be a path to an existing HTML file in the file system. 

<struct name="Local url" helpconfig="urlinst,800,640" helpdata="existing/path/to/a/html/file/index.html">
    ...
</struct>

HTML data

If you want to show your own message in the HTML window, you have to use the attribute helpconfig with the modifier code and assign a valid HTML code to the attribute helpdata. 

<struct name="Html data" helpconfig="code,400,100" helpdata="Help message">
    ...
</struct>


Figure 3: HTML help message.

HTML data with links in default browser

This case works like the previous with the only exception that the links appeared in the HTML message will be shown in the default browser. To do that you have to use the attribute helpconfig with the frontcode modifier. Note that the special characters must be written with the HTML codification. 

<struct name="Html data with link" helpconfig="frontcode,400,100" helpdata="&lt;a href=http://www.usc.es/&gt;USC web page&lt;/a&gt;">
    ...
</struct>

Tooltip

Other way to display on-site help is showing tooltips. The tooltips are messages that appear if you keep the cursor a few seconds over the widget header. This is possible with the attribute tooltip="message" where message is the help text you want to show. 

<leaf name="Tooltip" type="char" tooltip="This is a tooltip message">
</leaf>
