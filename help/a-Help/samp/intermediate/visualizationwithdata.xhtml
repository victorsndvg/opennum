Visualization with data

In this example you can see the modifier data and how to use it. It can be used in pointdata and celldata attributes. Both pointdata and celldata attributes always goes along plot attribute.

With data modifier you specify directly a field name to visualize it. This tutorial review some basic concepts from the Field visualization tutorial.

Pointdata and celldata

Below is shown an example of the use of the pointdata="data:[...]" atribute, being analog the use of celldata attribute. 

<struct name="Field" mesh="file:field3D.vtu" plot="filled"  pointdata="data:Active power density (W/m^3)">
</struct>
