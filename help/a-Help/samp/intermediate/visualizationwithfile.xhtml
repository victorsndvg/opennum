Visualization with file

In this example you can see the modifier file and how to use it. It can be used in mesh, add_mesh, data, showfile and source attributes.

With this modifier you specify directly a file path to show its content or visualize it. This tutorial extends some basic concepts from the Mesh visualization, Field visualization and Char list leafs tutorials.

Mesh

mesh is the attribute used to specify the mesh to represent, in both mesh and field representations the mesh file can be directly referenced with the file modifier as follows: 

<struct name="Mesh" mesh="file:mesh3D.mfm"  plot="mesh">
</struct>
<struct name="Field" mesh="file:field3D.vtu"  plot="mesh" pointdata="data:Active power density (W/m^3)" > 
</struct>

Add mesh

add_mesh is the attribute used to visualize fields and additional meshes simultaneously, you can use the file modifier as follows: 

<struct name="Field with additional mesh" mesh="file:field3D.vtu" add_mesh="file:mesh3D2.mfm" plot="filled"  pointdata="data:Active power density (W/m^3)">
</struct>

Data

data is the attribute used to visualize 2D charts, with this attribute you can use the file modifier as follows: 

<struct name="Show line chart" data="file:linechart.gr2" plot="2d_graph">
</struct>

Showfile

showfile is the attribute used to show the content of a plain text file, you can use the file modifier as follows: 

<leaf name="File content" type="charlist" showfile="file:file.txt"">
</leaf>

Source

source is the attribute used to create a single or multiple selection list from the content of a file, you can use the file modifier as follows: 

<leaf name="Selection single" type="charlist" source="file:file.txt">
</leaf>
<leaf name="Selection multiple" type="charlist" source="file:file.txt" selection="multiple">
</leaf>
