Actions

In this example we introduce a new tag, the action tag. With this tag you can launch or stop a executable process and clean the widgets and visualization areas.

The action tag can only appear as a unique submenu child. The name attribute of the action tag indicates the type of action to perform.

Run executable file

To run a executable file its mandatory the use of the action tag with the name="exec" attribute. The action tag can contain childs with the param tag, those elements are an ordered list of executables to run. 

<action name="exec">
	<param>executable.bat</param>
</action>

Stops executable file

To stop a process started from OpenNum, you can use the action tag with the name="kill_exec" attribute. This will kill the last process launched from OpenNum. 

<action name="kill_exec">
</action>

Close plots

It is also posible to close all the the open plots and clear de widget area with action tag. To get this action done you have to set the attribute name="close_plots". 

<action name="close_plots" />
