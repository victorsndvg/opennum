Menu

As a starting point of a series of tutorials, we begin with an example that shows the creation of the menu bar and the definition of the elements it contains.

This example shows the basis for creating the structure of the menu bar. Discuss the attribute name, mandatory for most tags, which defines the name and title of each item, and the attributes hidden and separator.

Application name

It's mandatory to use menus tag as the root of the XML hierarchy. The name attribute sets the application name displayed in the title bar. This example creates a menu of an application called Beginner: menu. 

<menus name="Beginner: menu">
    ...
</menus>

Menu bar

The tag menu allways appears as child of the tag menus. This tag allows to create items in the menu bar. This example creates a menu called Menu 1. 

<menu name="Menu 1">
    ...
</menu>

Submenus

The tag submenu sets the subitems shown when clicking on an menu bar item . This examples create a submenu called Submenu 1. 

<submenu name="Submenu 1">
</submenu>


Figure 1: Submenu.

The element submenu can contain other submenu elements and build a second level of submenus. The following code shows how to build a two level submenu. 

<menu name="Menu 2">
    <submenu name="Submenu 1: Level 1">
        <submenu name="Submenu 1: Level 2">
        </submenu>
    </submenu>
</menu>


Figure 2: Two level submenu.

Some common attributes

With the title attribute you can force to show a different title to the value asigned to the name attribute, this can be usefull to write titles in unicode.

With the hidden="true" attribute you can create item in the hierarchy that will not be shown, this can be usefull to define hide options to the final users.

If you want to separate some elements in different groups of menu or submenu you can use the attribute separator=true. This example creates a menu separator and a submenu separator, and uses the title and hidden attributes. 

<menu name="Menu 1">
    <submenu name="Submenu 1">
    </submenu>
    <submenu separator="true">
    </submenu>
    <submenu name="Submenu 2" title="Submenu 2: title">
    </submenu>
    </submenu>
    <submenu name="Hidden submenu" hidden="true">
    </submenu>
</menu>
<menu separator="true">
</menu>
<menu name="Menu 2">
</menu>


Figure 3: Menu and submenu separators.
