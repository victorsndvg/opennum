The application Beginner - mesh visualization shows different ways of visualizing a finite element mesh in OpenNum. It also explains the use of plot, mesh, celldata, pointdata and showvalues attributes. For the examples, we suppose that the file mesh.unv is in the current working folder and it has a valid mesh.

The XML file of this example can be found at <installdir>/apps/a-Beginner - mesh visualization/a-New/local.mnu.xml, where <installdir> is the installation folder.

Triangulation

This type of visualization allows us to represent the mesh as a wireframe (see Figure 1). This can be done by using a struct element with attributes plot="mesh" and mesh="prefix:filename", as it is explained in the XML syntax of Mesh visualization.

<struct name="Triangulation" plot="mesh" mesh="file:mesh.unv">
</struct>


Figure 1: Wireframe mesh.

Domain references

This type of visualization allows us to represent the nonzero domain references associated with the elements of the mesh (see Figure 2). A struct or leaf element must be accompanied by plot="references" and mesh="prefix:value" attributes. A celldata="data:element_ref" attribute indicates which field will be shown. The specific syntax is explained in Mesh visualization.

The showvalues="true" attribute allows us to display a list with the values achieved by the field associated with the mesh.

<leaf name="Domain references" type="charlist" plot="references" mesh="file:mesh.unv" celldata="data:element_ref"
showvalues="true"></leaf>


Figure 2: Domain reference selected.
