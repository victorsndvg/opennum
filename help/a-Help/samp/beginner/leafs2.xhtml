Char list leafs

The tag leaf is not only used for data input, may also be used to allow for single or multiple selections from a list of options.

This example shows the use and behavior of the new attributes selection and showfile along with the attribute type="charlist".

The selection attribute allows to select one or several option from a list displayed in a box or combo box. To create this list you can use the new tag element as a child of a leaf.

The showfile attribute shows the content of a plain text file in a text box.

Single selection

The tag leaf with the attributes type="charlist" and selection="single" display a combo box that allows to keep selected one between several options. selection="single" is the default behavior of the leaf of type="charlist". The sample code teaches how to build a widget of this type. 

<leaf name="Charlist" type="charlist" selection="single">
	<element>Element 1</element>
	<element>Element 2</element>
	<element>Element 3</element>
</leaf>

To write a evenly spaced list of floats you can use the Matlab® format (start:step:end).


Figure 1:Single selection widget.

Multiple selection

The tag leaf with the attributes type="charlist" and selection="multiple" display a box that allows to keep selected some options from a list. The sample code teaches how to build a multiple selection widget. 

<leaf name="Charlist" type="charlist" selection="multiple">
	<element>Element 1</element>
	<element>Element 2</element>
	<element>Element 3</element>
</leaf>


Figure 2:Multiple selection widget.

Show file

The tag leaf with the attributes type="charlist" and showfile="file:filename.txt" shows the content of a plain text file. This sample code teaches the use of the attribute showfile="file:filename.txt" 

<leaf name="plain text file" type="charlist" showfile="file:file.txt">
</leaf>


Figure 2:Plain text widget.
