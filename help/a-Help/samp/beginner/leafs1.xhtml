Leafs

The application Beginner - leafs is an example that shows how to create widgets to ask data trough the Data window. The leaf element allow us to request data to the user. The data type depends on the type attribute.

The XML file of this example can be found at <installdir>/apps/a-Beginner - leafs/a-New/local.mnu.xml, where <installdir> is the installation folder.

Integer/real data

A leaf element with type="float" attribute allows us to ask for integer or real scalars and arrays. The array components can be separated by blanks or commas; parentheses and square brackets are valid array delimiters. It also supports a valid MATLAB array using the colon operator as it is shown in Figure 1. The following example creates a widget to ask for integer/real data: 

<leaf name="float-variable" type="float">
</leaf>


Figure 1: Real array in Matlab.

Complex data

A leaf element with type="complex" attribute allows us to ask for complex scalars or arrays. The valid format is (float,float) (float,float) ..., as it is shown in Figure 2. The string is not processed if it does not meet the previous format. The following code creates widget to ask for complex data:

<leaf name="complex-variable" type="complex">
</leaf>


Figure 2: Introducing a complex array.

String data

A leaf element with type="char" attribute allows us to ask for a single string. The following code creates a widget of this type:

<leaf name="text-variable" type="char">
</leaf>

Filename data

A leaf element with type="file" attribute allows us to ask for a filename. This widget contains a button to display an Open file dialog:

<leaf name="file-path" type="file">
</leaf>
