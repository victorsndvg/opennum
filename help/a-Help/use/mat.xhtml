Materials database

The materials database can store the physical properties of a set of materials that are relevant for the applications.

The materials database could not appear if the distributor of the package decides that it is not necessary for his applications. When it appears, the materials database can accessed from the menu item Materials database, located in the right end of the menu bar (such position strengthen the fact that the materials database does not depend on a particular application).

The user can add, delete and modify the materials stored in the database. Every modification is recorded in a copy of the original database, located in the user HOME folder. The submenu item Reset restores the original database located in the installation folder. Please be aware that a modification in the materials database will affect to the existing working folders whenever their simulation is re-calculated.

The physical properties of each material and their definition depends on the package distributor. Please refer to the distributor's help to know the details.
