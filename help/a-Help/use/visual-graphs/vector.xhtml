Vector graphic

This graphic shows the vector field using arrow glyphs (see Figure 1). Two options can be modified in the Data window:

  • Scale: Sets the scale for the arrows length.
  • Density: Sets the relative density of the arrows.

The value of the field in a point can be shown by double-clicking in the Graphic scene.


Figure 1: Vector graphic.

Vector deformed graphic

This graphic deforms the underlying mesh using the vector field and it shows the modulus of the vector field on the deformed mesh (see Figure 2). The option Scale factor can be modified in the Data window: it sets the scale factor for the deformed mesh.


Figure 2: Vector deformed graphic.

Vector components

This graphic permits to show each a vector component using a filled graphic (see Figure 3). The option Vector component can be modified in the Data window: it sets the vector component to be shown.

If you prefer a more specific scalar visualization of the vector components, you can use formulas.


Figure 3: Vector component graphic.
