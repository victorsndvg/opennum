Vector field-related graphics

Vector graphic

This graphic shows the vector field using arrow glyphs (see Figure 1). Two options can be modified in the Data window (see Figure 10):

  • Scale: Sets the scale for the arrows length.
  • Density: Sets the relative density of the arrows.

The value of the field in a point can be shown by double-clicking in the Graphic scene.
