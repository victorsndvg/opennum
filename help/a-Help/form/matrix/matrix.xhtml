Description

The file extension for the format MTX is also .mtx.

The main features of this format are:

  • The file can store two dimensional arrays with any data length.
  • The file also stores strings defining row and colum labels.
Back to top

Variables

The variables needed to define a .mtx file are:

  • rownum is an integer with the number of rows;
  • colnum is an integer with the number of columns;
  • xi is a numeric or string array of length colnum containing a row;
  • rowlabeli is a string containing the label of the i-th row;
  • collabelj is a string containing the label of the j-th col;

Writing a .mtx file

Every .gr2 file must be written in ASCII and be composed of the following records:

  1. A first record containing rownum
  2. A record containing colnum
  3. a record containing xi with its components separated by blank spaces;
  4. A record, for each i in {1, ..., rownum}, row containing rowlabel;
  5. Finally, a record, for each i in {1, ..., colnum}, column containing rowlabel;
