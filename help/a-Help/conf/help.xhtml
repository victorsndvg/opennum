Help

You can distribute your own help with the package. Help files must be located in the folder <installdir>/help, where <installdir> is the installation folder.

All the subfolders in help will be shown as submenu items in the menu Help in alphabetical order. To change that order, the subfolders can be named as prefix-name, where name is the submenu's name and prefix is a string only used to alphabetize the submenus items. For example, creating the subfolders a-Getting Started and b-Applications Manual, Getting Started and Applications manual will appear as submenus, in that order.

Every subfolder must contain at least one HTML / XHTML file to act as an index of the rest of the files: OpenNum searches the names index.xhtml, index.html and index.htm, in that order. Its content is shown when the users selects the submenu item in the Help menu. To show them, OpenNum calls the application associated by the operating system to HTML files (usually, a web browser).
