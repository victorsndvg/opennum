Attribute tooltip

The elements struct and leaf can have the optional attribute tooltip="value", that indicates a string to appear as a tooltip when the user hovers the pointer over a widget, without clicking it.

<struct name="cond" title="Existing conditions:" tooltip="You can add, delete or edit the conditions">
  ...
</struct>

Attributes helpdata and helpconfig

The elements struct and leaf can have the optional attribute helpdata="value", that shows a new button Help in the widget. value can be an HTML code, an URL, or a local file. The optional attribute helpconfig="mode, width, height, title" configures the help window:

  • mode sets how to interpret value; there are three possibilities:
    • url indicates that value is an URL when it starts by http:// and a local file otherwise;
    • urlinst indicates that value is a file with a path relative to the installation folder;
    • code indicates that value is HTML code;
  • width and height set the width and height in pixels of the help window (set -1 to get the default values);
  • title sets the window title.

If the attribute helpconfig is missing, OpenNum takes helpconfig="url, -1, -1, Help".

<leaf name="one" type="float" helpdata="http://www.usc.es/">
</leaf>
<struct name="cuatro" helpdata="file.png" helpconfig="url,200,200,Image">
</struct>
<struct name="tres" helpdata="help/interface/doc.html" helpconfig="urlinst,300,200,Document">
</struct>
<leaf name="two" type="float" helpdata="<b>HTML Help!</b>" helpconfig="code,400,300,Help">
</leaf>
