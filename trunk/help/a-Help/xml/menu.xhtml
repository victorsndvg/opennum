Element menus

Every menu file or materials database file must contain a single menus element as a root element. Its children elements define the menu items of the menu bar. The attribute name="value" indicates that value is the name of the associated application. This will be the name shown in the title bar.

<menus name="Electrostatics">
  ...
</menus>

Element menu

Each element menu defines a menu item of the menu items of the menu bar. It must be a children of the element menus. The attribute name="value" indicates the text that will appear in the menu item. The values Project, Help and Materials database are forbidden since they are managed by OpenNum.

<menus name="Electrostatics">
  <menu name="Mesh">
    ...
  </menu>
</menus>
Back to top

Element submenu

Each element submenu defines a submenu item inside a menu item. It must be a children of an element menu. The attribute name="value" indicates the text that will appear in the submenu item.

<menu name="Properties">
  <submenu name="Materials">
    ...
  </submenu>
</menu>

Each element submenu can contain only one of the following elements struct, leaf or action.
