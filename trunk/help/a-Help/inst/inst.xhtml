Requirements

OpenNum is programmed in Python. These are the prerrequisites to properly use the application (v means "version"):

  • Python: Programming language, 2.5 ≤ v < 3.0; recommended, v2.7.2.
  • wxPython: WxWidget bindings for Python, v2.8.
  • Python-VTK: VTK bindings for Python, v ≥ 5.4; recommended, v5.8.
  • Paramiko: SSH2 protocol for Python, only necessary for remote connections.

The installation of these prerrequites can be done using binaries or from source.

Binaries for Debian-based Linux distributions

The previous prerrequisites can be downloaded from the official repositories. As an example, here we present how to install them using apt-get:

        sudo apt-get install python
        sudo apt-get install python-wxgtk2.8
        sudo apt-get install python-vtk
        sudo apt-get install python-paramiko

Binaries for Windows XP/Vista/7 and Mac OS

We recommended to install pythonxy, a scientific-oriented Python distribution. It contains all of the required packages by OpenNum, except Paramiko.

If you prefer to install them separately, these are the websites to download them:

  • Python: http://python.org/download/
  • wxPython: http://www.wxpython.org/download.php
  • Python-VTK: http://www.vtk.org/VTK/resources/software.html
  • Paramiko: http://bialix.com/python/

Binaries for Mac OS

The only method we have been tested is to use the Enthought Python Distribution.

Installation from source

If you prefer to build the prerrequisites from source, these are the websites to download them:

  • Python: http://www.python.org/download/releases/
  • wxPython: http://wxpython.org/download.php
  • Python-VTK: http://www.vtk.org/VTK/resources/software.html
  • Paramiko: http://www.lag.net/paramiko/
