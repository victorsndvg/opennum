Description

The format GR2 (Graph 2D) is related with the data files admited by the module TRACXX of Modulef. The file extension for the format GR2 is also .gr2.

The main features of this format are:

  • The file can store several graphs, each of them with different abscisas.
  • The file also stores strings defining the graph title, labels and legend.

Variables

The variables needed to define a .gr2 file are:

  • title is a string with the graph title;
  • xlabel is a string with the label for abscissas;
  • ylabel is a string with the label for ordinates;
  • n is a integer value with the number of graphics; for each i in {1, ..., n},
  • npi is a integer value with the total number of points that form the i-th graph;
  • xi is a double array of length npi containing the abscissas of the i-th graph;
  • yi is a double array of length npi containing the ordinates of the i-th graph;
  • legendi is a string with a legend description of the i-th graph.

Writing a .gr2 file

Every .gr2 file must be written in ASCII and be composed of the following records:

  1. A first record containing n; for each i in {1, ..., n},
    1. a record containing npi;
    2. a record containing xi with its components separated by blank spaces;
    3. a record containing yi with its components separated by blank spaces;
  2. A record containing title;
  3. A record containing xlabel;
  4. A record containing ylabel;
  5. Finally, for each i in {1, ..., n}, a record containing legendi.
