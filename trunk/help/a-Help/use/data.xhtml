The Data window is the part of the interface where data is introduced by the user. The user can introduce data by interacting with graphical elements called widgets. In this page, all the available widgets are presented.

Single data

Numbers

This widget allows us to enter real or complex data using a text box. The widget is shown in Figure 1. Input is recorded either when the Enter key is pressed or when another menu or widget is clicked.

In order to enter a real array, user must provide the elements separated by commas or blank spaces; they can be enclosed in parentheses or square brackets. The MATLAB colon operator is also supported.

In order to enter a complex scalar, user must provide its real and imaginary parts enclosed in parentheses; for example, type (a,b) to enter the scalar a+bi. To enter a complex array, user must type several complex scalars separated by commas or blank spaces; for example, type (a,b) (c,d) to enter the array [a+bi, c+di].


Figure 1: Widget for numbers.

Fixed lists

No selection

This widget displays a set composed of predefined elements. The elements can be either strings or the name of structures where additional information should be filled inside. The set is shown in a multiline list box. The widget is shown in Figure 4.


Figure 4: Widget for fixed lists with no selection.

Single selection

This widget allows us to select a single element from a set composed of predefined ones. The elements can be either strings or the name of structures where additional information should be filled inside. The set is shown in a drop down list box. The widget is shown in Figure 5.


Figure 5: Widget for fixed lists with single selection.

Multiple selection

This widget allows us to select several elements from a set composed of predefined ones. The elements can only be strings, not the name of structures. The set is shown in a multiline list box. The widget is shown in Figure 6.

To select several elements in GNU/Linux systems the Control key must be kept pressed.

User-defined items

This widget allows us to create a set with user-defined elements. The set is shown in a multiline list box and elements are added through a text box. The Add and Del buttons allows us to enlarge or reduce the set. The widget is shown in Figure 7.1.

Single selection
User-defined items

This widget allows us to create a set of user-defined elements and to select a single one. The set is shown in a combo box through which elements can be added or selected. To add a new element just type the name and press the Enter key. The Del button allows us to delete elements from the set. The widget is shown in Figure 8.1.
