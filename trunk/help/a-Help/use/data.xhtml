The Data window is the part of the interface where data is introduced by the user. In this window, a particular widget is presented for each data type.

Introducing real or complex data

When the user must introduce real or complex data, the widget shown in Figure 1 is used.


Figure 1: Widget for real data.

When a real array is expected, the input can be enclosed in parentheses or square brackets and the elements can be separated by commas or blank spaces. It also supports a valid MATLAB array using the colon operator.

To introduce complex scalars in the graphical interface, you have to type their real and imaginary parts using parentheses; for example, type (a,b) to give the scalar a+bi. To introduce complex arrays, type several complex scalars; for example, type (a,b) (c,d) to give the array [a+bi, c+di].

The input is recorded when the user either press Enter or visits another menu or widget.

File or folder data

To introduce files or folders, the widget shown in Figure 3 is used.


Figure 3: Widget for files or folders.

When the user press the button to the right, with an ellipsis, the widget calls the operating system-default dialog box to select files or folders. In the left box, the relative path to the last selected file or folder is shown.

Selecting a single string from a list

When the user must select a single string from a list, the widget shown in Figure 4 is used.


Figure 4: Widget for single selection.

This widget displays a list pressing the button to the right, with the triangle pointing down. Only one string can be selected.

Selecting multiple strings from a list

When the user must select multiple strings from a list, the widget shown in Figure 5 is used.


Figure 5: Widget for multiple selection.

This widget shows a list of strings. To select several strings in GNU/Linux, the Control key must be kept pressed.

Configurable lists

When the user can modify a list, the widget shown in Figure 6 is used.


Figure 6: Widget for configurable lists.

This widget shows a list of strings and several buttons to modify it. To add a string to the list, write it in the lower box and press the Add button. To delete an existing string, select it and press the Del button.

Back to top

Configurable lists with single selection

When the user can modify a list where a single item must be selected, the widget shown in Figure 7 is used.


Figure 7: Widget for configurable lists with single selection.

This widget shows a combo box where the user can write new items and record them pressing Enter. To delete an existing item, select it and press the Del button.

Visualization of plain text files

When an output written in a plain text file must be presented, the widget shown in Figure 8 is used.


Figure 8: Widget to show plain text outputs.

This widget is the only one that intended to show results, not to get inputs. The user can copy & paste the content to use it in another widget.
