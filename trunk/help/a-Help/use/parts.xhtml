Parts of the graphical interface

The graphical interface is composed of several parts, as we see in Figure 1:.


Figure 1: Parts of the graphical interface.

  • Title bar: contains the package title, the application name and the working folder.
  • Menu bar: contains the menu items, defined for the current application.
  • Data Window: shows the widgets that collect data.
  • Visualization Window: shows the resulting fields and graphs.
  • Log Window: shows messages from the interface and the called executable files.

The status bar is not relevant in OpenNum.
