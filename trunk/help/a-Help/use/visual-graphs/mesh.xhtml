Point numbering graphic

This graphic shows the global number of a selected point in the external surface of the mesh. This graphic can be accompanied by a real data widget to introduce an point number and to see the number of the point selected with the mouse (see Figure 6). The point numbers can be shown activating the Labels button in the Main bar.

Materials graphic

This graphic shows materials associated to the mesh domains (see Figure 8). This graphic can be available only when a Materials database exists. The materials legend can be shown activating the Table button in the Main bar.


Figure 8: Materials graphic.

Back to top

Slice mesh graphic

This graphic shows a slice of the mesh. It is only available for 3D meshes. Two options can be modified in the Data window (see Figure 9):

  • Origin: Changes the origin point of the plane.
  • Normal vector: Changes the normal vector of the plane.

The latter options can also be changed using the mouse in the Graphic scene. The plane can be hiding by deactivating the Plane button in the Main bar.


Figure 9: Slice mesh graphic.
