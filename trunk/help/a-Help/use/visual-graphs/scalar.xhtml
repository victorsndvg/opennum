Threshold graphic

This graphic shows a scalar field within a threshold. Two options can be modified in the Data window (see Figure 2):

  • Upper limits: Sets the upper limit of the threshold.
  • Lower limits: Sets the lower limit of the threshold.

The value of the field in a point can be shown by double-clicking in the Graphic scene.


Figure 2: Threshold graphic.

Back to top

Deformed graphic

This graphic shows the mesh deformed in the Z-axis by the scalar field values (see Figure 3). The option Scale in Z-axis can be modified in the Data window: sets the scale in the Z-axis. The field can be also shown colored in the domain using the Colored/Plain button in the Main bar.


Figure 3: Deformed graphic.

Isolines graphic

This graphic shows a scalar field using isolines (see Figure 4). This graphic is only available for 2D domains. Two exclusive options can be modified in the Data window :

  • Number of contours: Sets the number of contours in the graphic.
  • Contours at: Sets an array of values which the isolines will be plotted.


Figure 4: Isolines graphic.

Back to top

Plot over line

This graphic shows the scalar field along a line (see Figure 5). The complete field is shown in the left part of the Visualization window, together with the line. On the right part, the graph of the field along the line is shown. Two options can be modified in the Data window:

  • First point: Sets the coordinates of the beginning point of the line.
  • Second point: Sets the coordinates of the ending point of the line.

The position of the line can also be changed using the mouse in the Graphic scene.


Figure 5: Plot over line graphic.

Back to top

Slice graphic

This graphic shows a slice of the scalar field. It is only available for 3D domains. Two options can be modified in the Data window (see Figure 6):

  • Origin: Changes the origin point of the plane.
  • Normal vector: Changes the normal vector of the plane.

The latter options can also be changed using the mouse in the Graphic scene. The plane can be hiding by deactivating the Plane button in the Main bar. The value of the field in a point can be shown by double-clicking in the Graphic scene.


Figure 6: Slice graphic.
