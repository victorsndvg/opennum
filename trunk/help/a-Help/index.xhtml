OpenNum help

OpenNum is an application designed to distribute scientific solvers with a nice graphical interface.

Here we show the main topics of the OpenNum help:

  • Description

    Explains the capabilities of OpenNum.

  • License

    Presents the license conditions.

  • Installation

    Describes the prerequisites and the steps for installation.

  • Usage

    Explains how an user can work with the graphical interface.

  • Configuration

    Explains how a distributor can configure OpenNum.

  • XML syntax

    Describes the XML syntax for menu creation.

  • Sample apps

    Describes all the examples of use included with OpenNum.

  • Remote run

    Describes how to remotely run an executable file from OpenNum.

  • File formats

    Details the supported file formats.
