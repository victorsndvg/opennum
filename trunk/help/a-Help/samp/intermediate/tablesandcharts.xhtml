Tables and charts

This example teaches how to organize the leaf and struct elements to show tables and charts with OpenNum.

We discuss the new attribute show for the struct tags.

Tables

To represent single values or arrays of floats or complex in a table you have to use the attribute show="tabular" in a struct tag.

You have to define at least a leaf tag as child of the struct tag with the attribute show="tabular", this will create the Show button that displays the entered data in a table. 

<struct name="Table" show="tabular">
    <leaf name="Col 1" type="float">
    </leaf>
    <leaf name="Col 2" type="float">
    </leaf>
    ...
</struct>


Figure 1: Show button.

Charts

OpenNum also allows you to represent 2D charts. To do this you have to create at least two float leaf tags as child of the struct tag and use the attribute show="plot".

The first leaf child represents the abscissa axis and the following leaf tags represents different variables in the ordinate axis. The dimension of all the dependant variables must agree with abscissa dimension. 

<struct name="Line chart" show="plot">
    <leaf name="Abscissa" type="float">
    </leaf>
    <leaf name="Ordinate 1" type="float">
    </leaf>
    ...
</struct>

Tables and charts

If you want to show both tables and charts at the same time you can use the attribute show="all", with the same restrictions that the charts, as follows: 

<struct name="Line chart" show="all">
    <leaf name="Abscissa" type="float">
    </leaf>
    <leaf name="Ordinate 1" type="float">
    </leaf>
    ...
</struct>
