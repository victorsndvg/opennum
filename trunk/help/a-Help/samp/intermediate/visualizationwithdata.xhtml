Visualization with data

On this page:

  • Pointdata and celldata

The application Intermediate - visualization with data shows the modifier data and how to use it in pointdata and celldata attributes. With data modifier you specify directly a field name to visualize it. This tutorial review some basic concepts from the Field visualization tutorial.

The XML file of this example can be found at <installdir>/apps/n-Intermediate - visualization with data/a-New/local.mnu.xml, where <installdir> is the installation folder.

Pointdata and celldata

Below is shown an example of the use of the pointdata="data:[...]" atribute, being analog the use of celldata attribute. 

<struct name="Field" mesh="file:field3D.vtu" plot="filled"  pointdata="data:Active power density (W/m^3)">
</struct>
