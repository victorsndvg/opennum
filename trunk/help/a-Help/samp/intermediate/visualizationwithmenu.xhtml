Visualization with menu

The application Intermediate - visualization with menu shows the modifier menu used in mesh, pointdata, celldata, add_mesh, data, showfile and source attributes.

The XML file of this example can be found at <installdir>/apps/p-Intermediate - visualization with menu/a-New/local.mnu.xml, where <installdir> is the installation folder.

With this modifier you can dinamically specify different paths in the XML menu to show or visualize its content. This tutorial extends some basic concepts from the Mesh visualization, Field visualization and Char list leaves tutorials.

This tutorial assumes the existence of some paths in the xml hierarchy, to better understanding the use of this modifier we encourage to follow this tutorial with the corresponding XML example. The XML file of this example can be found at <installdir>/apps/o-Intermediate - visualization with menu/a-New/local.mnu.xml, where <installdir> is the installation folder.

Mesh

mesh is the attribute used to specify the mesh to represent, in both mesh and field representations the mesh file can be directly referenced with the menu:/path/to/a/mesh modifier, where /path/to/a/mesh must be an existing path to any type of leaf which contains a valid mesh or field fiel on the file system, as follows: 

<struct name="Mesh" mesh="menu:/path/to/a/mesh"  plot="mesh">
</struct>

Pointdata or celldata

pointdata or celldata attributes are used to define the field type and to give the name of a field contained in some file. You can use the menu:/path/to/a/field/name modifier, where /path/to/a/field/name must be a existing path to any type of leaf or struct which name represents a valid field in the mesh file, as follows: 

<struct name="Field" mesh="file:field3D.vtu" pointdata="menu:/path/to/a/field/name" plot="vector_field">
</struct>

Add mesh

add_mesh is the attribute used to visualize fields and additional meshes simultaneously, you can use the menu modifier as follows: 

<struct name="Field with additional mesh" mesh="file:field3D.vtu" add_mesh="menu:/path/to/a/mesh" pointdata="menu:/Visualization/Choose field/field">
</struct>

Data

data is the attribute used to visualize 2D charts, with this attribute you can use the menu:/path/to/a/chart modifier, where /path/to/a/chart must be an existing path to any type of leaf which represents a valid chart file on the file system, as follows: 

<struct name="From menu" plot="2d_graph" data="menu:/path/to/a/chart">
</struct>
Back to top

Showfile

showfile is the attribute used to show the content of a plain text file, you can use the menu:/path/to/a/text/file modifier, where /path/to/a/text/file must be an existing path to any type of leaf which represents a valid text file on the file system, as follows: 

<leaf name="File content" type="charlist" showfile="menu:/path/to/a/text/file">
</leaf>
Back to top

Source

source is the attribute used to create a combobox with selectable list from the content of a file, you can use the menu modifier as follows: 

<leaf name="Selection single" type="charlist" source="menu:/Materials database/Open/List">
</leaf>
<leaf name="Selection multiple" type="charlist" source="menu:/Materials database/Open/List" selection="multiple">
</leaf>

The path /Materials database/Open/List will be explained in the Materials and properties tutorial.
