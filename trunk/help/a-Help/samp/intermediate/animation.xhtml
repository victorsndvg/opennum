Animation

OpenNum gives you the possibility to visualize the time or frequency evolution of a field. This means that you can view animations with OpenNum.

To display an animation you must use PVD (Paraview Data) files in the mesh attribute, this is the default file format for ParaView visualizer. The field file set pointed by the PVD file must contain any VTK file type supported by OpenNum (polygonal, uniform rectilinear, nonuniform rectilinear, curvilinear, or unstructured).

In this example we introduce the new attribute evolution which speficies if the user will see time or frequency field sequencies. This attribute allways appear along the mesh, plot and poindata or celldata attributes.

Time animation

The default behavior of OpenNum while working with PVD files is to show a time animation bar. This behavior can be forced using the attribute evolution="time". 

<struct name="Time animation"  evolution="time" plot="vector_field" mesh="file:animation/animation.pvd" pointdata="data:Magnetic field">
</struct>


Figure 1: Time animation bar.

Frequency animation

To visualize frequency animations you have to use the attribute evolution="frequency". The difference with the time animations is that frequency animation do not allow to do data interpolation in the middle of two given timesteps. 

<struct name="Frequency animation"  evolution="frequency" plot="vector_field" mesh="file:animation/animation.pvd" pointdata="data:Frequency field">
</struct>


Figure 2: Frequency animation bar.
