Menus

The application Beginner - menus is an example that shows how to create a simple menu bar with menu and submenu items. It also introduces the name, hidden and separator attributes.

The XML file of this example can be found at <installdir>/apps/a-Beginner - menus/a-New/local.mnu.xml, where <installdir> is the installation folder.

The application name

Every XML menu file must contain a single menus element as the root of the XML hierarchy. The name attribute sets the application name, displayed in the title bar. The following example creates a menu for an application called Beginner: menus.

<menus name="Beginner: menus">
    ...
</menus>

Menu items

The menu element must always appear as child of the element menus. This element allows us to create items in the menu bar. The following lines create a menu called Menu 1.

<menus name="Beginner: menus">
    <menu name="Menu 1">
        ...
    </menu>
</menus>

Submenu items

The submenu element defines a submenu of a menu item. The following example creates a submenu called Submenu 1 as it is shown in Figure 1.

<menus name="Beginner: menus">
    <menu name="Menu 1">
        <submenu name="Submenu 1">
        </submenu>
            ...
    </menu>
</menus>


Figure 1: Submenu.

The submenu element can contain other submenu elements in order to form a second level of submenus. The following example shows how to build a two level submenu as it is shown in Figure 2. 

...
<menu name="Menu 2">
    <submenu name="Submenu 1: Level 1">
        <submenu name="Submenu 1: Level 2">
        </submenu>
    </submenu>
</menu>


Figure 2: Two level submenu.

Some common attributes

The title admits a UTF-8 string as a value; when it appears replace the name as the title of the associated widget. This attribute can be useful when we want to show non ASCII characters in the graphical interface.

The hidden="true" attribute creates a submenu item that will not be shown. This can be usefull to define parameters managed by the interface, but hidden to the users.

To classify the menu and submenu items in groups, the attribute separator=true will show a line between them.

The following example, shown in Figure 3, creates a menu separator and a submenu separator, and shows the use of title and hidden attributes.

<menu name="Menu 1">
    <submenu name="Submenu 1">
    </submenu>
    <submenu separator="true">
    </submenu>
    <submenu name="Submenu 2" title="Submenu 2: title">
    </submenu>
    </submenu>
    <submenu name="Hidden submenu" hidden="true" title="title with UTF-8 character: &#176;">
    </submenu>
</menu>
<menu separator="true">
</menu>
<menu name="Menu 2">
</menu>


Figure 3: Menu and submenu separators.
