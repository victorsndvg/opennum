Actions

On this page:

  • Running an executable file
  • Stopping a process
  • Closing visualizations

The application Beginner - actions is an example that shows how to run and stop a executable file through OpenNum. It also explains how to close all the visualizations in the Visualization window.

The XML file of this example can be found at <installdir>/apps/a-Beginner - actions/a-New/local.mnu.xml, where <installdir> is the installation folder.

Running an executable file

Opennum can run an external executable file by using the action element with name="exec" attribute. The action element must contain as many children param elements as executable files will be run.

<action name="exec">
    <param>executable.bat</param>
</action>

Stopping a process

A process can be stopped from OpenNum by using an action element with name="kill_exec" attribute. This code will stop the last process initiated from OpenNum.

<action name="kill_exec">
</action>

Closing visualizations

It is also possible to close all the visualizations in the Visualization window and clear the Data window by using an action element with name="close_plots" attribute.

<action name="close_plots" />
